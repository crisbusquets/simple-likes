A lightweight and theme-friendly Like button for WordPress posts and pages. One like per user. AJAX-powered. No bloat.

== Description ==

Simple Like Button is a lightweight plugin that adds an accessible, AJAX-powered "Like" button to your posts, pages, and archives. It’s fast, privacy-friendly, and fully theme-compatible.

**Features:**

- 👍 One like per user (logged-in or guest)
- ❤️ Live Like count
- ⚡ AJAX-powered, no page reloads
- 🎯 WordPress-native button styles
- 🧼 Toggle visibility (posts, pages, archives)
- 📊 Like stats dashboard with reset options
- 🔐 GDPR-compliant: no IPs, no fingerprinting, no personal data
- 🧩 No jQuery, no tracking, no bloat

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugin screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Go to **Settings > Simple Likes** to configure.

== Frequently Asked Questions ==

= Does this store any personal data? =  
No. The plugin uses WordPress user IDs or anonymous cookie IDs. No IP addresses or personal identifiers are stored.
